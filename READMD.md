# Landing Page – Sass Foundation Project 🌑

## Overview

This project was built during my early days of learning Sass.  
It’s a minimal landing page focused on structure, styling discipline, and clean organization.

Simple in appearance — foundational in impact.

## Focus ⚙️

- Sass architecture (partials, variables, nesting)
- Maintainable styling patterns
- Clean layout structure
- UI consistency

## Tech Stack

- HTML5  
- Sass (SCSS)  
- CSS3  

## Why It Matters

This project represents the stage where I moved from writing plain CSS to thinking in scalable styling systems.  

Every advanced skill starts with structured basics — this was one of mine. 🖤
